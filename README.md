# Docker Tutorial

It is a tutorial for learning docker containerization. Through this tutorial i can learn how to containerize a nodejs server as well as how to avoid unnecessary both files and folders using '.dockerignore' . I learned how to work with Dockefile using different commands.

# Installation

* Requirements

    - Docker 
    - Git
    - Node js && npm

* Installation

    - git clone https://github.com/Midhun-u/Docker-tutorial.git
    - Inside Docker-tutorial
    - npm install
    - docker build -t node-server ./
    - docker run node-server
